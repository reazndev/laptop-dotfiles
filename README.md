> **A dark, translucent, and minimalist setup for Sway.**

## Applications used

| Component | Application | Description |
| :--- | :--- | :--- |
| **Window Manager** | [Sway](https://swaywm.org/) | Tiling Wayland compositor |
| **Terminal** | [Kitty](https://swaywm.org/) | GPU-accelerated, supports transparency |
| **Status Bar** | [Waybar](https://github.com/Alexays/Waybar) | Highly customizable Wayland bar |
| **Menu/Launcher** | [Vicinae](https://www.google.com/search?q=https://github.com/loiccoyle/vicinae) | Fancy menu/launcher (Rofi alternative) |
| **Lock Screen** | [Swaylock](https://github.com/swaywm/swaylock) | Screen locker for Wayland |
| **Wallpaper** | [Awwww](https://www.google.com/search?q=https://github.com/LGF73/awwww) | Wallpaper daemon written in Rust |
| **IPC** | Swaymsg | Message sending for Sway |
